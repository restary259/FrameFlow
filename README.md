# FrameFlow

**FrameFlow** is an automatic, dynamic Minecraft performance mod for Fabric 1.21+ that adapts your game settings in real-time to stabilise FPS and reduce stutter. Designed for compatibility with Sodium, Indium, and standard modpacks.

## Features

- Adaptive FPS control and performance monitoring
- “Optimize Now” button for one-click cleanup
- Modular actuators (particles, render distance, entity LOD, chunk/unloader, etc.)
- Compatibility with Sodium, Indium, Fabric API
- In-game metrics & benchmarking tools
- Configurable via simple UI (Cloth Config / AutoConfig)

## Install

See the [Fabric wiki](https://wiki.fabricmc.net/) for Java and Loom setup instructions. Java 21 recommended for MC 1.21+ (unless other mods require Java 17).
